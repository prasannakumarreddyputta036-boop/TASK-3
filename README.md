# Task 3 - Responsive Web Development with JavaScript and Public APIs

## Overview

This project demonstrates advanced front-end development concepts by combining **Responsive Web Design**, **Interactive JavaScript**, and **External API Integration**. The application adapts to different screen sizes, includes interactive UI components, and fetches real-time data from a public API.

---

## Objectives

### 1. Responsive Web Design
- Create a responsive website that works on:
  - Mobile devices
  - Tablets
  - Desktop screens
- Use CSS Media Queries to adjust layouts, font sizes, spacing, and other UI elements.

### 2. Interactive JavaScript
- Build an interactive application using JavaScript.
- Implement either:
  - Interactive Quiz
  - Image Carousel

### 3. API Integration
- Learn how to fetch real-time data from a public API.
- Display the retrieved data dynamically on the webpage.

---

## Features

- Responsive layout using CSS Media Queries
- Modern and user-friendly interface

- Fetches real-time data from a public API
- Dynamic content rendering

## Responsive Design

The website uses CSS Media Queries to provide an optimal experience across different devices.

### Desktop
- Multi-column layout
- Larger font sizes
- Full navigation

### Tablet
- Adjusted spacing
- Medium font sizes
- Responsive layout

### Mobile
- Single-column layout
- Smaller font sizes
- Touch-friendly interface

---

## Interactive JavaScript

This project includes interactive JavaScript features such as:

- Event Listeners
- DOM Manipulation
- Dynamic Content Updates
- User Interaction Handling

---

## Public API Integration

The application fetches real-time information from a public API using the JavaScript Fetch API.

### Example APIs
- Joke API
- Weather API

The received JSON data is parsed and displayed dynamically on the webpage.
